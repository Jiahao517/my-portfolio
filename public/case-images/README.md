# 作品图片替换说明

直接覆盖这里的图片即可替换页面内容，刷新浏览器后自动生效。

文件夹对应页面：

- `dingtalk` -> `/dingtalk`
- `wencai` -> `/wencai`
- `chat-spec` -> `/chat-spec`
- `innovation` -> `/innovation`

编号对应页面中自上向下的图片顺序，例如：

- `dingtalk/06.png` 是钉钉页面第 6 张图片
- `wencai/03.png` 是问财页面第 3 张图片
- `chat-spec/01.png` 是智能对话设计规范页面第 1 张图片
- `innovation/10.png` 是设计创新页面第 10 张图片

替换时保持文件名不变，例如把新图片重命名为 `06.png` 后覆盖原文件。
目前四个页面固定读取 `.png` 文件。
