# Zhong Jiahao Portfolio

Standalone Next.js project for Zhong Jiahao's portfolio site.

## Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS v4

## Development

```bash
npm install
npm run dev
```

## Available Commands

```bash
npm run dev
npm run build
npm run start
npm run lint
npm run typecheck
npm run check
```

## Notes

- This repository is intended to be maintained as an independent project.
- Research artifacts and template-specific automation files were intentionally left out of this standalone copy.
